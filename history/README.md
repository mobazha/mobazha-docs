# Public History and Supersession Registry

This directory preserves public material that has been replaced, withdrawn, or
made historical. It is intentionally empty at initialization; content must not
be imported only because it exists in an old or private repository.

Every historical record must state:

- original title, date, author or source, and former lifecycle status;
- why it was replaced or withdrawn;
- the current authoritative replacement;
- whether any part remains valid;
- the date and reviewer who approved public archival;
- provenance showing that the original was already suitable for public access.

Historical files begin with a visible warning and must never be linked as the
authority for current capability, pricing, payment, legal, security, or
governance claims.

## Record template

```markdown
# Historical: Original title

- Status: Historical
- Original date: YYYY-MM-DD
- Archived: YYYY-MM-DD
- Former source: URL
- Replaced by: URL
- Public provenance reviewed by: role or public identity

> Historical material. Do not use as current behavior or policy.

## Supersession reason

## Material retained for context
```
