# Mobazha Architecture Decision Records

ADRs preserve durable decisions and their rationale. They are immutable after
acceptance except for status and supersession metadata; corrections use a new
ADR that links back to the original.

## Registry

| ADR | Title | Status | Date |
|---|---|---|---|
| [0001](./0001-markdown-content-authority.md) | Markdown files are the documentation content authority | Accepted | 2026-07-04 |
| [0002](./0002-public-knowledge-authority.md) | Mobazha Docs owns public knowledge | Accepted | 2026-07-04 |

Start from [`0000-template.md`](./0000-template.md). An accepted ADR describes
an intended decision but does not by itself prove runtime implementation or
release availability.
